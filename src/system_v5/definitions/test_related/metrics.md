## Manual evalutation metrics
### Correct by ground truth
- consistent with the ground truth
- consistent with the retrieved context
- Answers the question precisely 
- Answers the question completely

### Correct by context
- Not consistent with the ground truth
- consistent with the retrieved context
- Answers the question precisely 
- Answers the question completely

### Acceptable
- consistent with the retrieved context
- Answers the question precisely 
- Answer the question incompletely

### Unprecise 
- consistent with the retrieved context
- Answers the question imprecisely (and thus automatically incompletely)

### Incorrect 
- Not consistent with the ground truth
- Not consistent with the retrieved context


### Incorrect Abstain 
- Abstains when answer is answerable

### Correct Abstain 
- Abstains when answer is unanswerable
